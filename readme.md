# 🗂️ Vanilla JS Todo App - AI 기반 개발 기록

이 문서는 Vanilla JavaScript로 구현된 Todo 앱의 기능 개발 과정에서 사용한 AI 프롬프트와 코드 수정 내역을 기록한 문서입니다.

---

###  필수 미션 1 : 투두 CRUD 구현하기

#### 🤖 AI 코드 생성 프롬프트
```
// 직접 작성해주세요.

기존 코드에 상태별 필터링 기능을 추가해줘.

- 전체 / 진행 중 / 완료 탭을 클릭하면 해당 상태의 투두만 표시돼야 해
- 현재 선택된 탭은 시각적으로 구분되는 스타일이 적용되어야 해
```

#### 👨‍💻 AI 코드 수정한 부분
```
// 직접 작성해주세요.
예시) A 기능을 구현하기 위해 작성한 부분에서 키보드 엔터 입력 이벤트를 포함할 수 있도록 ~ 부분을 수정했습니다.
```
